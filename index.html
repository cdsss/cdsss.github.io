JS 基本教程

JS 简介

JavaScript 是脚本语言，是一种轻量级的编程语言，是可插入 HTML 页面的编程代码。

JS 基本功能

  • 写入 HTML 输出
  • 对事件作出反应
  • 改变 HTML 内容
  • 改变 HTML 图像
  • 改变 HTML 样式
  • 验证输入

JS 使用

HTML 中的脚本必须位于 <script> 与 </script> 标签之间。

脚本可被放置在 HTML 页面的 <body> 和 <head> 部分中。
